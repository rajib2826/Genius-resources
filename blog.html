The Document Object Model( DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and a document is penetrated and manipulated.

One website development, DOM is a Document Object Model. It's a programming interface that allows us to produce, change, or remove rudiments from a website document. DOM manipulation is when you use JavaScript to add, remove, and modify rudiments of a website.

3.What is Event Bubble?

Event bubbles are a concept in the Document Object Model. This occurs when an element receives an event, and bubbles up its parent and ancestor elements in the DOM tree until that event reaches the parent element.
